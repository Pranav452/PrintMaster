## Getting Started

1. Clone this bad boy:
   ```
   git clone https://github.com/Pranav452/PrintMaster.git
   cd PrintMaster
   ```

2. Install the goodies:
   ```
   npm install
   ```

3. Fire it up:
   ```
   npm run dev
   ```

4. Head to [http://localhost:3000](http://localhost:3000) and voila!
